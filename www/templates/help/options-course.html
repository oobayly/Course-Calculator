Triangle course The course fits into the box as defined by length × width.
 Angle #1 is the True Wind Angle (TWA) that the boat should see on the 1st down-wind leg.
Trapeziod course The course fits into the box as defined by length x width.
 Angle #1 is the True Wind Angle (TWA) that the boat should see on the 1st down-wind leg. Angle #2 is the True Wind Angle (TWA) that the boat should see on the 3rd down-wind leg.
Windward-leeward with Spreader A windward-leeward course with a port-hand spreader mark at 90° TWA. The spreader distance is defined by the course width.
Committee vessel Declination is the local difference between True North and Magnetic North. Negative values indicate Magentic North is West of True North.
This app expects all bearings to be provided respective to Magnetic North.
Declination can be entered manually, or calculated using NOAA World Magnetic Model (WMM) for the current committee vessel's location. Magentic Declination (Wikipedia) The committe vessel can be placed at specified proportion of the beat. If this is 0%, the start line pin-end mark is also the leeward laid mark.
